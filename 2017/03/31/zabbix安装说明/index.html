zabbix安装说明

Server安装

使用Yum安装。

  1. 导入源

    # rpm -ivh http://repo.zabbix.com/zabbix/3.2/rhel/7/x86_64/zabbix-release-3.2-1.el7.noarch.rpm

  2. 安装zabbix包

    yum install zabbix-server-mysql zabbix-web-mysql

  3. 安装MySQL数据库

    CentOS 7的yum源中没有正常安装mysql时的mysql-sever文件，需要去官网上下载。

成功安装之后重启mysql服务。
