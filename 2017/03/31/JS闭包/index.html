JS闭包

发表于 2017-03-31 | 更新于 2017-06-04 | 分类于 JavaScript原理

闭包的特性

  1. 函数嵌套函数
  2. 函数内部可以引用外部的参数和变量
  3. 参数和变量不会被垃圾回收机制回收

使用闭包主要是为了设计私有的方法和变量。

  • 优点：是可以避免全局变量的污染。
  • 缺点：是闭包会常驻内存，会增大内存使用量，使用不当很容易造成内存泄露。
